Sample Work

Remediation Report Sample

A representative extract from a client accessibility audit report, demonstrating the issue-by-issue format with WCAG criterion mapping and concrete fix code.

Page / Location Issue Description WCAG SC Severity Recommended Fix
Contact Page — Section Header Heading tag skips level: <h3> used directly after <h1>, breaking document outline 1.3.1 High <h2>Contact Us</h2>
Resources Page — PDF Download Link Image link has no alt text; screen readers announce URL string only 1.1.1 Critical alt="Download Annual Report PDF"
Homepage — Navigation Menu Dropdown menu not keyboard operable; cannot be opened or dismissed with keyboard 2.1.1 Critical aria-expanded + keydown handler
Sign-up Form — Email Field Input has no visible label; placeholder used as sole label disappears on input 1.3.1 High <label for="email">Email</label>
Dashboard — Status Alerts Error messages conveyed by colour alone; users with colour blindness receive no indication 1.4.1 High role="alert" + text indicator
Blog Post — Body Text Text on grey background fails contrast ratio: measured 3.2:1 (required minimum 4.5:1) 1.4.3 High color: #767676 → #595959
